    Star-Delta/Delta-Star Transformation

    Star-Delta Transformations and Delta-Star Transformations allow us to convert impedances connected together in a 3-phase configuration from one type of connection to another.

    Configuration Converter
    Delta-Star Converter

    To convert a delta network to an equivalent star network, we implement the formula below in relation with the image above:

    RA = R1 * R2 / (R1 + R2 + R3)
    RB = R1 * R3 / (R1 + R2 + R3)
    RC = R2 * R3 / (R1 + R2 + R3)
    Star-Delta Converter

    To convert a star network to an equivalent delta network, we implement the formula below in relation with the im3ge above:

    R1 = (RA * RB / RC) + RA + RB
    R2 = (RA * RC / RB) + RA + RC
    R3 = (RB * RC / RA) + RB + RC
